# Detecting Pneumonia from Chest X-Rays
High-performance pneumonia classifier examining DeiT + CNNs, custom loss functions, and reproducible training pipeline for chest X-Ray analysis.

**TL;DR:** Trained multiple models (DeiT, DenseNet, EfficientNet, MobileNet, ResNet, ShuffleNet, Ensemble) on a 3-class chest X-ray dataset. **Best model:** DeiT-Base Patch16 224 at **87% accuracy**.
